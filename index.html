How It Works

세 단계로 승부를 읽는다

실시간 배당 데이터를 수집해 패턴을 감지하고, ML 앙상블로 최종 예측을 도출합니다.

01
📡

배당 실시간 수집

Playwright 기반 자동 스크래퍼가 OddsPortal에서 15개 이상의 해외 북메이커 배당(개장·마감)을 수집합니다.

02
🔍

패턴 감지 & 앙상블

Mirror, Palindrome, Alternating Pairs 등 8가지 시계열 패턴을 북메이커 투표 방식으로 감지합니다.

03
🤖

ML 최종 예측

Random Forest 모델이 패턴 신호와 배당 방향 투표를 결합해 승팀과 신뢰도를 최종 산출합니다.

Features

주요 기능

단순 통계를 넘어 시장 참여자의 심리와 정보 흐름까지 포착합니다.

📊

배당 방향 투표 (BM Dir Vote)

개장 대비 마감 배당 변화 방향을 북메이커별로 집계해 시장 컨센서스를 도출합니다.

🔄

8종 시계열 패턴 인식

Mirror, Tail Mirror, Palindrome, Repeat Blocks 등 배당 수열의 구조적 패턴을 자동으로 감지합니다.

🌐

멀티 북메이커 앙상블

단일 북메이커에 의존하지 않고 15개 이상의 소스를 앙상블해 노이즈를 줄입니다.

⚡

GitHub Actions 자동화

매일 22:30 KST에 배당 수집·예측 파이프라인이 자동 실행되어 항상 최신 상태를 유지합니다.

📈

백테스트 & 검증 로그

경기 결과가 확정되면 자동으로 정확도를 기록하고 누적 성과를 트래킹합니다.

🎯

신뢰도 기반 PASS 필터

신뢰도가 낮은 경기는 PASS 처리해 무분별한 예측을 지양하고 정밀도를 높입니다.

Backtesting

과거 데이터로 검증된 패턴

실제 배팅이나 추측이 아닌, 과거 경기 배당 데이터를 기반으로 패턴의 유효성을 먼저 검증했습니다.

🗄️

과거 배당 수집

2026 시즌 개막 이후 KBO 전 경기의 개장·마감 배당 데이터 수집

🔎

패턴 소급 적용

과거 배당 수열에 8종 패턴 알고리즘을 소급 적용해 신호 생성

⚖️

실제 결과 대조

패턴 기반 예측값을 실제 경기 승패와 비교해 패턴별 적중률 산출

🔁

모델 지속 개선

축적된 검증 로그로 임계값·가중치를 조정하며 모델을 반복 개선
